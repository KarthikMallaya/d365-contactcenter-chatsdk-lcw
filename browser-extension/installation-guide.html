Installation Guide

Follow these simple steps to install and configure the Dynamics 365 Contact Center Chat Widget extension in Microsoft Edge or Google Chrome. The entire process takes less than 5 minutes. This is an internal Microsoft demo tool.

1

Download the Extension

Click the download button above to get the extension ZIP file. Save it to a location you can easily find, like your Downloads folder.

File Information

The ZIP file contains all necessary files for the extension including configuration, scripts, and styling.

2

Extract the ZIP File

Right-click the downloaded ZIP file and select "Extract All" (Windows) or double-click (Mac). Extract it to a permanent location where you won't accidentally delete it.

Important

Don't delete the extracted folder! Chrome needs access to these files for the extension to work. Keep them in a safe, permanent location.

3

Open Browser Extensions

Open Microsoft Edge or Google Chrome and navigate to the extensions page:

  • Microsoft Edge: Type edge://extensions/ in the address bar
  • Google Chrome: Type chrome://extensions/ in the address bar
  • Or click the browser menu (⋮) → Extensions
Microsoft Edge: edge://extensions/ Google Chrome: chrome://extensions/
4

Enable Developer Mode

In the top-right corner of the Extensions page, you'll see a toggle switch labeled "Developer mode". Click it to turn it ON (it should turn blue).

Why Developer Mode?

Developer mode allows you to install extensions from your computer instead of the browser's extension store. It's completely safe for this purpose.

5

Load the Extension

After enabling Developer mode, new buttons will appear. Click the "Load unpacked" button, then browse to and select the folder you extracted in Step 2.

Select the Right Folder

Make sure to select the folder that contains the manifest.json file. This is typically the "browser-extension" folder from the ZIP file.

6

Configure Your Settings

Click the extension icon in your browser toolbar (you may need to pin it first). This will open the configuration popup where you'll enter your chat widget settings:

Organization ID Your Dynamics 365 Org ID
Organization URL Your CRM URL
Widget ID Your chat widget ID
Company Name Optional - for logo branding
Questions URL Optional - Power Automate endpoint
Agents URL Optional - Dynamic agents endpoint

Auto-Detection

If you don't provide a company name, the extension will automatically use the domain of the website you're visiting for logo branding.

7

Start Using the Extension

Once configured, visit any website and you'll see a floating chat button in the bottom-right corner. Click it to open the premium chat widget!

Features

• Premium sidebar chat interface with 28% width
• Automatic logo detection from website domain
• Voice input with microphone support
• Camera and file upload capabilities
• Multiple AI agents to choose from
• Current page URL passed to chat context

Need Help?

If you encounter any issues during installation, please contact your Microsoft representative or refer to the README.md file included in the extension package.
